Student

Many different programs exist to store, sort and create different lists of students of schools. Students can be examined in many different ways through the characteristics they hold.

Assignment:
You will be examining a school of students over the next two labs. In this lab, you will be focusing on the Student class. Students have first and last names, ID numbers, GPAs, and expected graduation years. Their GPAs are calculated from an array of chars.

1. Fill in the missing parts of the constructor without parameters to read in the necessary information to create a Student object.

Read in the first name, last name, and expected graduation year of the Student from the user using Scanner class methods and save them in their corresponding instance variables. The Scanner class methods are found in its API.
Remember that you need the length of an array to initialize it. After the grades have been read in as a String and trimmed, use some examples to see how the length of the String, grades, with spaces in between correspond to the number of grades or chars within that String. After looking at a few examples, you should see a pattern emerge. Also, remember the difference between integer and double division. The number of chars is the length of the gradesList array.
Fill the char array gradesList. Pull each char out of the String, grades, and put it in gradesList. You may use the String method charAt for this.

2. Complete the constructor with parameters by setting the instance variables to the values in the parameters.

Use the keyword this to help differentiate between instance variables and parameter variables.
Don't forget to call the helper method, calcGpa method to set gpa. Notice calcGpa has a private modifier because it is only used internally as a helper method. public access is not needed. Assume that it is working as intended so that you can use it freely.

3. Fill in the getter methods for this class. There are 5 private instance variables, so there should be 5 getter methods.

4. Fill in the calcGpa method, making sure to fill in the type returned and the method body.

To calculate the GPA, you will need to traverse through the given array grades and find the average, given where A has a value of 4, B has a value of 3, C has a value of 2, D has a value of 1, and F has a value of 0. Other chars are not valid, so if another char is encountered, a warning should be printed and the program should be exited using a System method, which can be found in its API.

5. Complete the assignEmailAddress method to assign students a String email. A student's email is formatted as follows: the first letter of the student's first name followed by the student's last name, the student's expected graduation year, and @serrahs.com.

6. Write the equals method to determine whether two students are equal to each other. Two students are equal if their first and last names are the same and if they have the same ID number.

Remember, you cannot always use == to compare to Strings. The String class methods are found in its API.
