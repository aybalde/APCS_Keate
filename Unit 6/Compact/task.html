When arrays are created, we sometimes want to examine them with and without certain values. It is also important to know how to take arrays in methods as parameters and return them. This lab practices both of these skills.

Assignment:
Write a program that has a method that takes in an array of integers and returns a new array with all of the zeroes removed. The program will create the original array in the tester class rather than in the constructor. Examine the following arrays.

Original Array:

1 0 2 0 0 5

The new array resulting from running the method removeZeroes would be the following.

Array returned by removeZeroes:

1 2 5

1. Code the removeZeroes method to take in an array and return a new array with the zeroes removed.
Remember to declare a new array of ints and return that array. To help find the size of the array, you will need a loop to count the zeroes. You will not know the length of the array in the parameter, so you will need to use the variable length here.
Remember to use a loop fill the new array with all values but zeroes.
2. Create an array and fill it with the following values:
0, 1, 3, 0, 4, 6, 0, 0, 13, 25, 0

3. Run the removeZeroes method, and save the resulting array. Then, print the original array and the new array resulting from the removeZeroes method. Test your method with many different arrays to make sure it works as intended.

Remember the removeZeroes method is static, so it needs to be called on the class, not the object.
Remember when a method returns something, you need to create a variable to save it to if you want it saved. Otherwise, it will be returned and immediately picked up by the trash collector.
4. Make sure your name is commented out near the top of your source code.
