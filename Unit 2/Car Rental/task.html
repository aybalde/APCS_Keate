Car Rental

When you rent a car from an agency, the car is coded in the company's system with specific information like the following: license plate, make, model and year of car, and usually a special code. Here is an example of an algorithm to calculate a rental code using the information in the following way.

Rental Code = The first 3 characters of the car's make followed by the first 3 characters of the car's model, the last 2 digits of the car's year, a dash, followed by the sum of the ASCII code value for each char of license plate and the final 3 digit number of the license plate. You can assume cars makes and models will all be words with lengths of at least 3 and license plates will all be formatted as three letters followed by a three digit number. You can also assume all cars were made after the year 2010.

For example, if a car had the following information, the rental code would be as follows:

Make: Toyota
Model: Highlander
Year: 2019
License Plate: ABC123
Rental Code: TOYHIG19-320

The code's final 3 digit number was calculated using these steps:
1. Find the ASCII code values for A, B, and C: 65, 66, and 67.
2. Add those together with the 3 three digit number of the license plate: 65 + 66 + 67 + 123 = 320
