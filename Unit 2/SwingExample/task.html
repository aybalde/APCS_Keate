Swing Example

Java Swing is a graphical user interface (GUI) toolkit that has buttons, checkboxes, and more. It allows you to make well formatted interfaces the user can interact with. This exercise shows an example of a JFrame and clickable button created using classes and methods of the Swing class.

Assignment:
1. Explore the code to examine the code used to create a button that prints, "Hello World!" when you click it.
2. Create a new button that says, "Click Me!"

You will need to create a new JButton with the text, "Click Me" and set the bounds and size for it. Be sure your new button has a new name (otherwise the other one will not work). If needed, you can access JButton's API.

3. Code the new button so that when it is clicked, it prints "Goodbye!" when clicked. Play around with font, style, and color to format the printed message in the way you want. Be sure to actually change these components to see how Swing works! If you do not change all of these components, the Check button will print "Incorrect."
You will need to create a a new JLabel and and set the bounds and size for it. You will also need to set its font, style, size, and color. Be sure your new label has a new name (otherwise the other one will not work). If needed, you can access JLabel's API and Font's API. Your label creation will not yet create the "Goodbye!" message.
You will need to create a a new ActionListener that tells the computer to print out "Goodbye!" when the new button is pushed. Be sure your new ActionListener has a new name (otherwise the other one will not work). You will add the ActionListener to the button you created in Step 1. If needed, you can access ActionListner's API.
4. Add the JButton and the JLabel to the frame.
5. Test that both buttons work as they should and that examine the difference between the two messages. Is that what you expected given the changes you made in the message?
6.. Make sure your name is commented out near the top of your source code.
