Quadratic

Every quadratic equation has 2 roots. When the equation is in the form y = ax2+ bx + c, these roots can be found using the following algorithms:
x1 = (-b + √(b2 - 4ac))/(2a)
x2 = (-b - √(b2 - 4ac))/(2a)

Java expresses imaginary numbers as NaN, meaning not a number. You can assume that is an acceptable response for imaginary roots.
