Assignment:
Write a program to calculate a person's taxes when given their marital status and income.

1. Fill in the constructor with parameters to initialize the instance variables. Trim the String status to make sure there are no leading or trailing spaces. Then, assign the instance variable married to true if status holds "married" (in upper or lowercase) and false if status holds "single" (in upper or lowercase). Any invalid status is dealt with in the given else statement. Then, initialize taxableIncome to income. Any invalid income is dealt with in the given if statement below.
Remember to save the result of trimming status so that you can use it!
2. Create the necessary getter methods to make sure the instance variables are accessible.

3. Write the methods, calcSingleTax and calcMarriedTax, to calculate and return the taxable income of a person with the given marital status and income.
Note the modifiers on these methods are private. These are helper methods - they help achieve a different task within the class TaxPayer but are not used or accessible in any other class.
4. Write the method, calcTax, to calculate and return the taxable income of any person.
You will need to use your helper methods and if-statements to achieve this!
5. Verify that the program gives the taxes for people with the following tax information. Please be sure to enter these people in this order or your program will not test correctly.

Marital Status: Single
Income: $100,000
Marital Status: Married
Income: $200,000

6. Make sure your name is commented out near the top of your source code.
