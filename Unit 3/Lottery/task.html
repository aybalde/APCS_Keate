Lottery

The CA Lottery has many different games you can play to win money. You buy in with a set price on an initial ticket, and you can either pick a set of numbers or get assigned them randomly. The price per ticket and amount of numbers on your ticket is determined by the game you choose to play. In general, the higher the cost of the ticket, and the more numbers you have to choose, the higher the prize amount.

Assignment:
Write a program to pull the random winning combination of for a lottery game. In this specific game, there are 6 total numbers that must be pulled, and each number is between 1 and 100. Numbers can be chosen twice. The order of the numbers on the winning ticket must match the order of the randomly chosen winning combination.

1. Fill in the constructor without parameters to initialize the lottoNumbers. This constructor will randomly choose the winning number and can be used for customers who want to have the ticket machine choose a random combination for them.

Remember to save lottoNumbers as a String with spaces in between each number chosen. There should be no leading or trailing spaces.
Be sure to remember to use Math class methods found in its API to randomly choose the numbers on the customer's ticket.

2. Fill in the constructor with parameters to initialize the customer's lotto ticket. This constructor will allow customers to choose their own combination.
Remember to save lottoNumbers as a String with spaces in between each number chosen. There should be no leading or trailing spaces.
3. Create the necessary getter method to make sure lottoNumbers is accessible outside of the Lottery class.

4. Override the method, equals, to check to see if two Lottery objects are equal. The method should take a Lottery object in as a parameter and should return true if the two lotto numbers match, false if they do not.
Remember order matters here, so the numbers will have to match completely.
5. Write the method checkTicket to print the whether or not the customer's ticket is a winner. The method should take a Lottery object in as a parameter.
You should use the equals method you just wrote in this method.

6. First, enter the main method into the LotteryClient class. Then, verify that the program works by creating a Lottery object customerTicket and another Lottery object winningCombo and running the checkTicket method in the LotteryClient class. The output window should clearly display the customer's lottery ticket combination and the winning combination as well as whether or not the ticket is a winner.

7. Make sure your name is commented out near the top of your source code.
